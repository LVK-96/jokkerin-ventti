# Jokkerin Ventti

A workout interval timer app for Jokkerin Ventti, featuring a high-performance 3D visualization engine.

[![Deploy to GitHub Pages](https://github.com/LVK-96/jokkerin-ventti/actions/workflows/deploy.yml/badge.svg)](https://github.com/LVK-96/jokkerin-ventti/actions/workflows/deploy.yml)

**[▶️ Launch App](https://lvk-96.github.io/jokkerin-ventti/)**

> **Note:** The animation library is currently a **Work In Progress (WIP)**.

## Features

- [Jokkerin Ventti](http://mii.iki.fi/story/54/jokkerin-ventti) Workout timer.
- Real time rendered workout animations.

## Development

### Prerequisites

- **Node.js** 20+
- **Rust** (nightly toolchain required)
- **wasm-pack** (`cargo install wasm-pack`)

### Setup

1.  Install Node dependencies:
    ```bash
    npm install
    ```

2.  Start the development server (automatically compiles WASM):
    ```bash
    npm run dev
    ```

3.  Open <http://localhost:5173/jokkerin-ventti/>

### Build

To build the production version (compiles Rust to optimized WASM and builds the Vite app):

```bash
npm run build
```

Output is in `dist/`.

## Asset Pipeline

- Animations have been generated by **[HyMotion 1.0](https://github.com/Tencent-Hunyuan/HY-Motion-1.0)**.
- To reduce the size, FBX files from HyMotion are converted to custom binary format with blender python scripts in `tools/` directory.
- The covereted files are stored in `src/assets/animations`.

## Tech Stack

- **Frontend**: TypeScript, Vite
- **Core Engine**: Rust, WebAssembly
- **Graphics**: wgpu, WebGPU (WebGL fallback)
